### 7个数据源的原始格式



| 来源 | 格式 | 获取方式 |
|-------|-------|-------|
| Wikipedia | HTML表格 → DataFrame | pd.read_html() |
| yfinance | pandas DataFrame | yf.download() 批量 |
| SEC EDGAR XBRL | JSON (深层嵌套) | HTTP GET, 无需key |
| SEC EDGAR Submissions | JSON | HTTP GET |
| FRED | pandas Series | fredapi 库 |
| Ken French | 字典内含DataFrame | pandas_datareader |
| SEC CIK映射 | JSON | HTTP GET |
